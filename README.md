# Geological Data Dashboard (SQL + Report Builder)

**Данный проект предназначен для демонстрации SQL-логики и подходов к автоматизации отчётности без раскрытия корпоративных данных.**

---

## Overview

Проект автоматизации геологических данных и формирования отчетности на основе SQL и Microsoft Report Builder.

Решает задачу замены ручного ведения таблиц на автоматизированную систему с динамической фильтрацией, валидацией и контролем качества данных.

---

## Architecture

- **Source tables** → геологические и лабораторные данные  
- **SQL layer** → обработка, агрегация, QA/QC  
- **Report Builder** → визуализация и пользовательские параметры  

---

## Purpose

- Автоматическое формирование структурной копии вручную поддерживаемой таблицы геологических проб  
- Обновление данных по мере поступления новых записей  
- Гибкая фильтрация и валидация данных  

---

## Input Parameters

Отчёт формируется на основе пользовательских параметров:

- Месторождение  
- Участок  
- Тип выработки  
- Год  
- Месяц (поддерживается множественный выбор)  

**Скважины:**
- становятся доступны при наличии данных по выбранным параметрам  
- пользователь может выбрать одну или несколько скважин  

---

## Output Data

Результатом является табличный дашборд, содержащий:

- идентификатор скважины  
- идентификатор и тип пробы  
- интервалы отбора (от, до, длина интервала)  
- массу пробы  
- информацию о последней лабораторной поставке:
  - дата отправки  
  - номер задания  
- результаты лабораторного анализа (например, Au, Ag)  
- связи QA/QC:
  - родительская проба  
  - дубликаты  
  - стандарты  
- служебные поля для:
  - условного форматирования  
  - валидации данных  

---

## SQL Implementation

В проекте реализованы:

- CTE (Common Table Expressions) для поэтапной обработки данных  
- оконные функции:
  - `ROW_NUMBER()` — выбор последнего лабораторного результата  
  - `COUNT() OVER` — выявление дубликатов  
- временные таблицы и индексы для оптимизации  
- `UNION ALL` для объединения различных типов проб  
- сложные JOIN между:
  - скважинами  
  - пробами  
  - лабораторными результатами  

---

## Report Builder Integration

Дашборд реализован в Microsoft Report Builder и включает:

- основной dataset (финальный SQL-запрос)  
- дополнительные datasets для параметров  
- фильтрацию данных на уровне SQL-запроса
